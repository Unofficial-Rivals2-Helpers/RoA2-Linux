# RoA2-Linux
This is a central directory for all tools / information useful for running Rivals of Aether 2 on Linux Systems

## Cloning Repo:
git clone --recurse-submodules https://github.com/WK-Kz/RoA2-Linux.git

## Tools Included Here
- [Color Swap](https://github.com/WK-Kz/Color-Swap-ROA-2)
    - This is a fork of the original Color Swap tool provided by Keryan-666
- [Optimization Scripts](https://github.com/WK-Kz/RoA2-Optimization-Scripts)
    - Check this repo out which optimizes your game.
